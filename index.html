Competetive Analysis: Portfolios

Site Name Frog
Site URL www.frogdesign.com
Date looked at: 19.10.2012
Brand: Frog Designs web page gives the visitors an impression that they are playful and freshly-thinking, but a serious business as well.
Functionality For functionality you should talk about whether it effectively shows off what needs to be shown. If it is site with details can you see the details. Are there enough secondary pages showing off the design? Should there be annotations or commentary. Functionality can blur into navigation at times.
Interactivity For interactivity you should mention and show the sort of interactivity on the site. Do they use coverflow, lightbox and if so how elegantly? Do they allow sorting, categorizing (again this is like navigation).
Navigation Navigation for us will mean getting around different pages. When you are on a different page do you know where you are, how to get back, what the page is, what else there is?
